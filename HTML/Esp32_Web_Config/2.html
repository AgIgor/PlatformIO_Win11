Pinagem do ESP32 DEVKIT V1 DOIT (36 pinos)

Observações importantes: O ESP32 opera com 3.3V. Não aplique 5V diretamente aos pinos. Nem todos os pinos são criados iguais. Alguns têm funções específicas e não devem ser usados para outros propósitos.

Pinos de Alimentação

Pino Função
Vin Entrada de tensão (5V)
GND Terra (Ground)
3V3 Saída de 3.3V regulada

Outros Pinos

Pino Função
EN Chip Enable. Mantido em HIGH para o ESP32 funcionar.

Pinos a Evitar

Os seguintes pinos são usados internamente e devem ser evitados para uso geral: GPIO6, GPIO7, GPIO8, GPIO9, GPIO10, GPIO11.
