Pinos do ESP32 DEVKIT V1 DOIT de 36 pinos

GPIO Estado Funções Específicas Notas Selecione a Função
GPIO0 0 Boot Mode Selection (BOOT), Input, Output Deve ser mantido em LOW para entrar no modo de bootloader.
GPIO1 0 TX0 (UART0 Transmit) Saída de depuração padrão; usado durante upload de código.
GPIO2 0 Input, Output, LED_BUILTIN (em alguns modelos) Conectado a LED em algumas placas.
GPIO3 0 RX0 (UART0 Receive) Entrada de depuração padrão; usado durante upload de código.
GPIO4 0 PWM, I2C (SDA), ADC, DAC, Input, Output Multifunção.
GPIO5 0 PWM, I2C (SCL), ADC, Input, Output Multifunção.
GPIO6-11 0 - Reservados para comunicação com a memória flash (SPI Flash).
GPIO12 0 PWM, ADC, Input, Output Multifunção.
GPIO13 0 PWM, ADC, Input, Output Multifunção.
GPIO14 0 PWM, ADC, Input, Output Multifunção.
GPIO15 0 PWM, ADC, Input, Output Multifunção.
GPIO16 0 Input, Output Multifunção.
GPIO17 0 Input, Output Multifunção.
GPIO18 0 SPI (SCK), PWM, Input, Output Multifunção; frequentemente usado para SPI.
GPIO19 0 SPI (MISO), PWM, Input, Output Multifunção; frequentemente usado para SPI.
GPIO21 0 I2C (SDA), PWM, Input, Output Multifunção; frequentemente usado para I2C.
GPIO22 0 I2C (SCL), PWM, Input, Output Multifunção; frequentemente usado para I2C.
GPIO23 0 SPI (MOSI), PWM, Input, Output Multifunção; frequentemente usado para SPI.
GPIO25 0 DAC1, PWM, ADC, Input, Output Multifunção; pode ser usado como DAC.
GPIO26 0 DAC2, PWM, ADC, Input, Output Multifunção; pode ser usado como DAC.
GPIO27 0 PWM, ADC, Input, Output Multifunção.
GPIO32 0 ADC, Touch, PWM, Input, Output Multifunção; suporta detecção de toque capacitivo.
GPIO33 0 ADC, Touch, PWM, Input, Output Multifunção; suporta detecção de toque capacitivo.
GPIO34 0 ADC, Input Apenas entrada analógica.
GPIO35 0 ADC, Input Apenas entrada analógica.
GPIO36 0 ADC, Input (VP) Apenas entrada analógica; entrada do ADC1.
GPIO39 0 ADC, Input (VN) Apenas entrada analógica; entrada do ADC1.
